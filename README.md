# Online-QuizTest-generator
A Dynamic Web-Based Quiz Creation and Evaluation System


# Project Overview
The Online Quiz Generator is a dynamic, user-friendly web application that allows users to create, publish, take, and evaluate quizzes directly in the browser. The project is designed using HTML, CSS, and Vanilla JavaScript, focusing on interactivity, clean UI, and client-side data management.

This project demonstrates practical implementation of DOM manipulation, event handling, local storage, and dynamic rendering, making it suitable for academic lab work and beginner-to-intermediate web development learning.
# Objectives

To design an interactive quiz creation platform

To allow users to attempt quizzes and receive instant results

To implement automatic scoring and answer validation

To demonstrate real-world use of JavaScript logic and browser storage

| Technology           | Purpose                                    |
| -------------------- | ------------------------------------------ |
| **HTML5**            | Page structure and layout                  |
| **CSS3**             | Styling, animations, and responsive design |
| **JavaScript (ES6)** | Application logic and interactivity        |
| **LocalStorage**     | Persistent quiz data storage               |
| **JSON**             | Quiz data structure                        |

# Key Features

✅ Create quizzes with:

Multiple Choice

Multiple Answer

True / False questions

✅ Add, delete, and modify questions dynamically

✅ Select correct answers with validation

✅ Save quizzes using browser LocalStorage

✅ Load sample quizzes automatically

✅ Attempt quizzes with real-time answer selection

✅ Automatic score calculation and percentage grading

✅ Detailed result analysis with correct & incorrect answers

✅ Responsive and modern UI design



# How the System Works

Create Quiz

Enter quiz title and description

Add questions and select answer types

Save quiz to browser storage

Take Quiz

Select any available quiz

Answer questions interactively

Submit quiz when finished

View Results

Score calculated automatically

Percentage and performance message displayed

Answer review section highlights correct & incorrect responses
